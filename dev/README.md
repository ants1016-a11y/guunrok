# 개발 스크립트

## run.sh

게임 실행 전 자동 검사를 수행하는 스크립트입니다.

### 사용법

```bash
./dev/run.sh
```

### 수행 작업

1. **black 포맷터 확인 및 설치**
   - black이 설치되어 있지 않으면 자동으로 설치합니다.

2. **코드 포맷팅**
   - `black main.py`를 실행하여 코드를 자동 포맷팅합니다.

3. **문법 검사**
   - `python -m py_compile main.py`를 실행하여 문법 오류를 검사합니다.
   - 문법 오류가 발견되면 실행을 중단합니다.

4. **게임 실행**
   - 모든 검사를 통과하면 게임을 실행합니다.

### 주의사항

- **절대 `python main.py`로 직접 실행하지 마세요.**
- 항상 `./dev/run.sh`를 사용하여 실행하세요.
- 이렇게 하면 실행 전 자동으로 코드 품질을 검사할 수 있습니다.
