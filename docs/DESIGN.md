# 구운록 (Guunrok) - 게임 디자인 및 규칙

## 🎮 게임 시스템 개요

구운록은 턴제 카드 배틀 게임으로, 플레이어는 손에 든 카드(무공)를 사용해 적과 전투를 벌입니다.

## 👤 플레이어 스탯

### 기본 스탯
- **HP (체력)**: 100 (최대)
- **Defense (방어도)**: 0 (기본값)
- **Energy (행동력)**: 5 (최대, 턴마다 회복)

### 스탯 설명
- **HP**: 생명력. 0이 되면 패배
- **Defense**: 받는 데미지를 감소시킴. 공식: `실제 데미지 = 적 데미지 - Defense`
- **Energy**: 무공을 사용하는 데 필요한 자원. 턴 시작 시 최대치로 회복

## 🃏 카드 시스템

### 카드 타입
1. **ATTACK (공격)**: 적에게 데미지를 입힘
2. **DEFENSE (방어)**: 플레이어의 방어도를 증가시킴
3. **SKILL (기술)**: 다양한 보조 효과
4. **DEBUFF (약화)**: 적의 능력을 약화시킴

### 현재 구현된 카드

#### 삼재공 (SKILL)
- **비용**: 1 행동력
- **효과**: 
  - 50% 확률: 행동력 3 회복
  - 50% 확률: 방어도 3 증가
- **설명**: "행동력 3 회복 또는 방어도 3 증가"

#### 육합권 (ATTACK)
- **비용**: 2 행동력
- **효과**: 적에게 8-12 랜덤 데미지
- **설명**: "기본 공격으로 8-12 데미지"

#### 복호장 (ATTACK)
- **비용**: 4 행동력
- **효과**: 적에게 18-25 랜덤 데미지
- **설명**: "강력한 공격으로 18-25 데미지"

#### 포천삼 (DEFENSE)
- **비용**: 2 행동력
- **효과**: 방어도 8-12 랜덤 증가
- **설명**: "방어도 8-12 획득"

#### 유운지 (DEBUFF)
- **비용**: 3 행동력
- **효과**: 적의 공격력을 5 감소 (다음 턴부터 적용)
- **설명**: "적의 공격력을 5 감소시킴"

### 덱 구성
- 각 카드 2장씩, 총 10장
- 시작 시 손패 5장 뽑기
- 손패는 최대 5장까지 보유 가능

## ⚔️ 전투 시스템

### 턴 구조
1. **플레이어 턴**
   - 행동력 최대치로 회복
   - 손패가 5장 미만이면 카드 뽑기
   - 카드 선택 및 사용
   - 턴 종료 버튼으로 적 턴으로 전환

2. **적 턴**
   - 적이 자동으로 행동
   - 플레이어 턴으로 전환

### 데미지 계산 공식

#### 플레이어가 받는 데미지
```
실제 데미지 = max(0, 적 공격력 - 플레이어 방어도)
최종 HP = 현재 HP - 실제 데미지
```

#### 적이 받는 데미지
```
실제 데미지 = 카드 데미지 (랜덤 범위)
최종 HP = 현재 HP - 실제 데미지
```

### 방어도 시스템
- 방어도는 데미지를 감소시킴
- 방어도가 공격력보다 높으면 최소 0 데미지
- 방어도는 턴이 지나도 유지됨 (카드로 제거되거나 덮어쓰기 전까지)

## 👹 적 시스템

### 적 계급별 스탯

#### 녹림 졸개 (Minion)
- **HP**: 30
- **공격력**: 5
- **특징**: 기본 공격만 사용

#### 녹림 행동대장 (Elite)
- **HP**: 60
- **공격력**: 10
- **특징**: 
  - 기본 공격
  - 30% 확률로 강공격 (공격력 x2)

#### 녹림왕 (Boss)
- **HP**: 120
- **공격력**: 15
- **특징**:
  - 기본 공격
  - 40% 확률로 필살기 (공격력 x3)

### 적 AI 패턴
- 기본적으로 플레이어를 공격
- Elite 이상은 확률적으로 강한 공격 사용
- Debuff가 걸리면 공격력이 감소 (다음 턴부터)

## 🎯 승리/패배 조건

### 승리 조건
- 적의 HP가 0 이하가 되면 승리
- 승리 시:
  - 승리 화면 표시
  - "체력 회복 후 계속" 버튼으로 보상 받기
  - 체력 20 회복
  - 다음 적 생성

### 패배 조건
- 플레이어의 HP가 0 이하가 되면 패배
- 패배 시:
  - 게임 오버 화면 표시
  - "다시 시작" 버튼으로 게임 리셋

### 죽었을때 
계승 요소: 죽더라도 **'깨달음(경험치)'**은 남아서, 다음 시작 때 가지고 있던 기본 무공 카드가 강화된 상태로 시작한다.
설정 녹이기: "육체는 잊었으나, 영혼에 새겨진 검로는 기억한다."

## 📊 밸런스 조정 가이드

### 카드 밸런스
- **비용 대비 효과**: 비용이 높을수록 강한 효과
- **랜덤 요소**: 일부 카드는 랜덤 데미지/효과로 변동성 제공
- **시너지**: 방어 카드와 공격 카드의 조합

### 적 밸런스
- **난이도 곡선**: 졸개 → 행동대장 → 녹림왕 순으로 강해짐
- **HP 비율**: 플레이어 최대 HP(100) 대비 적 HP 비율
  - 졸개: 30% (30/100)
  - 행동대장: 60% (60/100)
  - 녹림왕: 120% (120/100)

### 전투 밸런스
- 플레이어는 탱커 스타일 (높은 HP, 낮은 공격)
- 적은 공격 중심 (높은 공격력, 낮은 HP)
- 방어도 시스템으로 플레이어 생존력 보장

## 🔧 기술적 세부사항

### 카드 사용 조건
- 행동력이 카드 비용 이상이어야 사용 가능
- 손패에 카드가 있어야 함
- 전투 중이어야 함 (battle_state == "playing")

### 카드 뽑기 시스템
- 턴 시작 시 손패가 5장 미만이면 자동으로 뽑기
- 덱이 비면 버린 카드 더미를 섞어서 덱으로 사용

### 상태 관리
- `battle_state`: "playing", "victory", "defeat"
- `turn_phase`: "player", "enemy"
- 전투 중이 아니면 카드 사용 불가
